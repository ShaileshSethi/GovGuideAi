# 🧭 GovGuide AI

> **One AI Assistant for Every Government Service.** 

GovGuide AI is an intelligent platform designed to simplify complex government procedures for citizens. By converting dense bureaucratic requirements into simple, actionable steps, it eliminates confusion around document gathering, application procedures, and government services.

## ✨ Features

- **Natural Language Search**: Describe your problem (e.g., *"I lost my Aadhaar card"*) and let AI figure out the exact service you need.
- **Personalized Action Plans**: Get step-by-step guidance, estimated processing times, and fee breakdowns.
- **Interactive Document Checklist**: Keep track of mandatory and optional documents needed for your application.
- **History & Caching**: All generated plans are saved locally so you never lose your progress.
- **Premium UI/UX**: A modern, glassmorphic design system tailored specifically for a "GovTech" aesthetic, ensuring trust and clarity.
- **100% Official Links**: Directly routes users to verified government portals.

## 🚀 Quick Start

### Prerequisites
- Node.js (v18.17.0 or higher)
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ShaileshSethi/NEW.git
cd NEW
```

2. Install dependencies:
```bash
npm install
```

3. Set up your environment variables:
Create a `.env.local` file and add any required API keys (e.g., for AI generation or Auth).

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router, Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4) + Vanilla CSS Variables for Theme Management
- **Icons & UI:** Heroicons & Glassmorphism effects
- **Authentication:** NextAuth (Auth.js)

## 🎨 Design Philosophy

GovGuide AI ditches the clunky "2010 government portal" look. Instead, it uses a **GovTech Green** palette (`#15803D`, `#F8FAF8`) combined with modern UI principles (Glassmorphism, hover-animations, and spacious typography) to build a trustworthy and memorable AI product.
